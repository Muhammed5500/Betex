# btx-monad

Paper-faithful implementation of
[BTX](https://category-labs.github.io/category-research/BTX-paper.pdf)
(Batch Threshold Encryption — Agarwal, Das, Gilkalaye, Rindal, Shoup) on
Monad testnet, packaged as an MEV-resistant DEX.

Built for the Monad Kayseri 2026 hackathon.

## Status

Work in progress. See the commits for build order.
